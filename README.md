# Juris Tati Marketplace

Marketplace pessoal com a versao corrigida (v0.2.0) do plugin juris-adv-os.

Base: plugin juris-adv-os v0.1.0 (Sbroggio Advocacia & IA Combativa),
licenca MIT — modificado conforme auditoria tecnica de 05/07/2026
(ver CHANGELOG.md dentro do plugin).

## Instalar

Cowork -> Settings -> Plugins -> Adicionar marketplace -> URL deste repositorio.
Depois: instalar o plugin `juris-adv-os` e abrir sessao nova.
